> **"Just as DNA forms the fundamental instructions for life, code forms the building blocks of the digital realm."**

A high-performance, glassmorphic developer portfolio merging genetic algorithmic visual art with robust modern technology. This project features a custom-built 3D DNA helix engine rendered on an HTML5 Canvas, designed for deep user immersion and seamless interaction.

---

## ✨ Key Features

- **3D DNA Helix Engine**: A custom mathematical coordinate system rendering a dynamic double helix with real-time scroll-velocity-based rotation.
- **Glassmorphic UI**: Premium aesthetic using backdrop-filter blur, CSS variables, and semi-transparent layers for a modern "frosted glass" look.
- **Interactive Parallax**: 3D tilt effects on project cards and a "lerping" custom cursor that tracks mouse movement with smooth lag.
- **Performance Optimized**: Built with vanilla JS and optimized Canvas rendering, achieving 60FPS animations while maintaining high efficiency.
- **Smart Filtering**: Dynamic project showcase with category-based filtering for Frontend, Creative, and Fullstack work.

---

## 🛠️ Tech Stack

### Frontend & Logic
* **Core**: HTML5, CSS3 (SCSS), JavaScript (ES6+).
* **Visuals**: HTML5 Canvas 2D API, GSAP for smooth physics-based animations.
* **Architecture**: Intersection Observer API for scroll-reveal logic and optimized asset loading.

This portfolio transcends typical static interfaces. It focuses on **dynamic interaction**, where every design choice is calculated down to the pixel.
